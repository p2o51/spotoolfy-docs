# Spotoolfy 文档

> 版本：1.3.1 | 最后更新：2024年12月

欢迎来到 Spotoolfy 文档中心！Spotoolfy 是一款使用 Flutter 开发的 Spotify 辅助客户端，为你带来歌词显示、AI 翻译、笔记记录、音乐洞察和时光机等功能，全部采用 Material Design 3 风格设计。

## 核心特性

- **实时歌词** - 多来源歌词获取（网易云、QQ音乐、LRCLIB），支持逐行滚动和手动搜索
- **AI 翻译** - 三种翻译风格（忠实、诗意、机器翻译），由 Gemini 驱动
- **笔记系统** - 为每首歌曲添加笔记和评分，记录你的音乐故事
- **时光机** - 重温往年今日收藏的歌曲，发现被遗忘的音乐回忆
- **音乐洞察** - AI 分析你的听歌习惯，生成个性化音乐人格
- **海报生成** - 为歌词、笔记、专辑创建精美的分享海报
- **动态主题** - 根据专辑封面自动生成配色方案
- **多语言** - 支持英语、简体中文、繁体中文、日语

## 版本历史

### v1.3 主要更新
- 新增时光机功能（今日历史回忆）
- 新增稀疏索引系统优化资料库扫描
- 新增 AI 对话功能（询问歌曲相关问题）
- 新增歌词、笔记、专辑海报生成
- 新增歌曲信息分析页面
- 新增网易云音乐原生翻译支持
- 深色模式支持
- 抄歌词模式优化
- 多语言支持扩展（繁体中文、日语）

### v1.2 主要更新
- 三种翻译风格（忠实、诗意、机器翻译）
- Gemini AI 洞察生成
- 搜索与资料库功能
