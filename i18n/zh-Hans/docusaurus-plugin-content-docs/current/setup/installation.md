# 安装、社群、联系方式

## 安装指南

### Android 安装

#### 方式一：Google Play 封闭测试（推荐）

1. 加入 [Google Groups 测试群组](https://groups.google.com/g/spotoolfytest)
2. 访问 [Google Play 商店页面](https://play.google.com/store/apps/details?id=com.gojyuplusone.spotoolfy.spotoolfy_flutter)
3. 点击「安装」

:::note
你必须先加入 Google Groups 才能访问封闭测试版本。
:::

#### 方式二：直接下载 APK

1. 加入 [Discord 社区](https://discord.gg/v56yDs6vRN)
2. 在 #releases 频道下载最新 APK
3. 在手机上安装 APK（可能需要允许「未知来源」安装）

### iOS/iPadOS 安装

1. 访问 [TestFlight 测试链接](https://testflight.apple.com/join/h2GR2Gbf)
2. 如果没有安装 TestFlight，App Store 会提示你安装
3. 在 TestFlight 中接受测试邀请并安装 Spotoolfy

### Web 版本（Beta）

- 访问 https://app.spotoolfy.gojyuplus.com/
- Web 版本功能受限：
  - 无法控制远程播放设备
  - 仅支持 Web API 功能
  - 可以查看歌词、记录笔记、使用翻译

## 系统要求

| 平台 | 最低版本 |
|------|---------|
| Android | Android 6.0 (API 23) |
| iOS | iOS 12.0 |
| Web | 现代浏览器（Chrome、Firefox、Safari、Edge） |

## 问题反馈

遇到问题时，请优先：
1. 查看 [已知问题和 Q&A](../advanced/faq)
2. 在 Discord 的 `#feedback` 或 `#bug-reports` 频道发帖
3. 在 GitHub 提交 Issue

反馈时请提供：
- 使用的设备和系统版本
- Spotoolfy 版本号
- 问题的详细描述和复现步骤
- 如果可能，附上截图或录屏
