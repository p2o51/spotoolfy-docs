# 漫游页面

漫游页面是你的音乐笔记中心，包含时光机、随机回顾和完整的笔记列表。这里记录着你与音乐的所有故事。

## 时光机

时光机让你重温往年今日收藏的歌曲，发现被遗忘的音乐回忆。

### 今日历史
- 自动查找往年今天（±1天容差）添加到收藏的歌曲
- 显示「X 年前的今天」标签
- 展示歌曲名、歌手、专辑封面

### 时光机交互
- **点击歌曲**：打开回忆详情，显示完整信息
- **播放**：重新播放这首往年的歌曲
- **查看全部**：进入完整的时光机页面

### 时光机页面
点击「查看全部」或日期范围按钮进入时光机页面：
- 显示所有符合条件的回忆歌曲
- 支持自定义日期范围查询
- 点击日期按钮切换「今日历史」和「自定义范围」模式

### 稀疏索引
时光机使用稀疏索引技术优化性能：
- 首次使用会扫描你的 Spotify 资料库
- 建立索引后查询速度极快
- 索引数据存储在本地，不会上传

## 随机回顾

随机展示你过去写的笔记，帮助你重温与音乐相关的记忆。

### 自动轮播
- 每 4 秒自动切换显示不同的笔记
- 显示笔记内容、歌曲信息、评分

### 回顾卡片
卡片显示：
- 笔记内容（如果有）
- 歌词快照（如果有）
- 歌曲名、歌手名
- 专辑封面
- 评分图标
- 记录时间

### 交互操作
- **点击卡片**：打开笔记详情
- **长按卡片**：立即播放该歌曲

## 轮播区域

时光机和随机回顾组成一个可滑动的轮播区域：
- **第一页**：时光机（今日历史）
- **第二页**：随机回顾
- 底部有页面指示器
- 左右滑动切换

## 所有笔记

页面下方显示你记录的所有笔记，按时间倒序排列。

### 搜索笔记
使用搜索栏搜索笔记：
- 支持搜索笔记内容
- 支持搜索歌曲名
- 支持搜索歌手名

### 筛选功能
使用筛选标签快速筛选笔记：

| 筛选项 | 说明 |
|--------|------|
| **全部** | 显示所有带内容的笔记 |
| **🔥 喜欢** | 只显示评分为「喜欢」的笔记 |
| **😐 一般** | 只显示评分为「一般」的笔记 |
| **👎 不喜欢** | 只显示评分为「不喜欢」的笔记 |
| **⭐ 仅评分** | 只显示没有文字内容的评分记录 |

每个筛选标签旁边显示对应的笔记数量。

### 笔记卡片
每条笔记显示：
- **笔记内容**：你写的文字（如果有）
- **歌词快照**：记录时的歌词（如果有）
- **歌曲信息**：歌名、歌手、专辑封面
- **记录时间**：显示为「今 HH:mm」「昨 HH:mm」或「MM-DD HH:mm」
- **评分图标**：👎 / 😐 / 🔥

### 笔记操作

#### 点击笔记
打开笔记详情面板，可以：
- 查看完整笔记内容
- 修改评分
- 编辑笔记内容
- 播放歌曲（从记录的时间戳位置）
- 生成分享海报
- 删除笔记

#### 滑动操作（窄屏模式）
- **向右滑动**：播放歌曲
- **向左滑动**：删除笔记（需确认）

### 统计信息
筛选栏左侧显示笔记总数，让你了解记录了多少音乐故事。

## 刷新数据

下拉页面可以刷新数据：
- 重新加载笔记列表
- 更新时光机数据
- 刷新随机回顾

## 响应式布局

漫游页面根据屏幕宽度调整布局：

### 窄屏（手机）
- 单列笔记列表
- 支持滑动操作（左滑删除、右滑播放）
- 更紧凑的卡片设计

### 宽屏（平板/桌面）
- 双列瀑布流布局
- 卡片网格展示
- 分隔线装饰

## 空状态

如果你还没有记录任何笔记：
- 显示友好的空状态提示
- 引导你在「正在播放」页面添加第一条笔记
