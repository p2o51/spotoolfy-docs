# 已知问题和 Q&A

## 已知问题

### 歌词相关

#### 歌词显示不正确
- **现象**：歌词时间戳与音乐不同步，或歌词内容错误
- **原因**：第三方歌词来源可能存在错误
- **解决**：使用手动搜索功能，选择其他版本的歌词

#### 某些歌曲没有歌词
- **现象**：显示「找不到歌词」
- **原因**：歌曲可能是小众曲目，或歌曲信息不完整
- **解决**：尝试手动搜索，使用不同的关键词组合

#### 翻译加载缓慢
- **现象**：点击翻译后等待较长时间
- **原因**：Gemini API 响应时间受网络影响
- **解决**：耐心等待，翻译结果会缓存，下次访问秒加载

### 播放控制

#### 设备列表为空
- **现象**：找不到任何 Spotify 设备
- **原因**：没有活跃的 Spotify 客户端
- **解决**：
  1. 在任意设备上打开 Spotify 客户端
  2. 播放一首歌曲激活设备
  3. 在 Spotoolfy 中刷新设备列表

#### 播放控制无响应
- **现象**：点击播放/暂停没有反应
- **原因**：可能是 Token 过期或网络问题
- **解决**：
  1. 检查网络连接
  2. 尝试重新登录 Spotify
  3. 确保 Spotify 客户端正在运行

#### 切歌后进度条不同步
- **现象**：进度条显示与实际播放位置不一致
- **原因**：网络延迟导致状态更新滞后
- **解决**：等待几秒，进度会自动同步

### 时光机

#### 首次使用时光机很慢
- **现象**：第一次打开时光机需要等待较长时间
- **原因**：正在建立稀疏索引，扫描 Spotify 资料库
- **解决**：耐心等待，索引建立后查询会非常快

#### 时光机没有显示内容
- **现象**：显示「没有找到回忆」
- **原因**：往年今日可能没有收藏过歌曲
- **解决**：尝试使用自定义日期范围查询更大的时间跨度

### iOS 特有问题

#### 登录后无法返回应用
- **现象**：完成 Spotify 授权后停留在浏览器
- **原因**：URL Scheme 回调可能被系统拦截
- **解决**：手动切换回 Spotoolfy 应用

#### TestFlight 版本过期
- **现象**：应用无法打开，提示版本过期
- **原因**：TestFlight 构建有有效期限制
- **解决**：等待新版本发布，或联系开发者获取更新

## 常见问题解答

### 账户与登录

**Q: 需要 Spotify Premium 吗？**
A: 完整功能需要 Premium。免费用户可以查看歌词、记录笔记，但无法控制播放。

**Q: 我的 Spotify 数据安全吗？**
A: Spotoolfy 不存储你的密码。通过 OAuth 授权只获取必要的权限，Token 安全存储在本地设备。

**Q: 可以同时登录多个账户吗？**
A: 目前只支持单账户登录。如需切换账户，请先登出当前账户。

### 歌词与翻译

**Q: 为什么有些歌词是乱码？**
A: 可能是歌词源编码问题。尝试手动搜索其他来源。

**Q: 翻译要花钱吗？**
A: Google AI API Key 可以免费获取和使用，个人使用完全免费。

**Q: 可以保存翻译后的歌词吗？**
A: 翻译会自动缓存在本地，也可以使用复制功能保存。

**Q: 翻译风格有什么区别？**
A:
- Faithful：直译，准确传达原意
- Melodramatic Poet：意译，更有诗意和情感
- Machine Classic：模拟早期机翻，风格独特
- NetEase：使用网易云原生翻译

### 笔记与数据

**Q: 笔记会同步到云端吗？**
A: 目前笔记只存储在本地设备。可以使用导出功能备份数据。

**Q: 换手机后笔记会丢失吗？**
A: 需要手动导出再导入。建议定期备份数据。

**Q: 可以删除所有数据吗？**
A: 可以在设备存储中清除应用数据，或卸载重装。

**Q: 笔记有字数限制吗？**
A: 没有硬性限制，但建议保持简洁。

### 功能与兼容性

**Q: 支持 Apple Watch 吗？**
A: 目前不支持，但可以通过 Spotoolfy 控制 iPhone/iPad。

**Q: 可以离线使用吗？**
A: 歌词和翻译需要网络。已缓存的内容可以离线查看，但播放控制需要联网。

**Q: 支持播客吗？**
A: 目前主要针对音乐优化，播客可能无法获取歌词。

**Q: 为什么没有 Android 小部件？**
A: 小部件功能在计划中，将在后续版本添加。

## 错误代码参考

| 错误代码 | 含义 | 解决方案 |
|---------|------|----------|
| `AUTH_EXPIRED` | 授权已过期 | 重新登录 Spotify |
| `NO_ACTIVE_DEVICE` | 没有活跃设备 | 在 Spotify 客户端播放任意歌曲 |
| `NETWORK_ERROR` | 网络错误 | 检查网络连接 |
| `LYRICS_NOT_FOUND` | 歌词未找到 | 尝试手动搜索 |
| `API_RATE_LIMITED` | API 请求过于频繁 | 稍后重试 |
| `TRANSLATION_FAILED` | 翻译失败 | 检查 API Key 配置 |
