# Spotoolfy へようこそ

> バージョン：1.3.1 | 最終更新：2024年12月

Spotoolfy ドキュメントセンターへようこそ！Spotoolfy は Flutter で開発された Spotify コンパニオンクライアントです。歌詞表示、AI翻訳、ノート記録、音楽インサイト、タイムマシン機能を提供し、すべて Material Design 3 スタイルでデザインされています。

## 主な機能

- **リアルタイム歌詞** - 複数ソースから歌詞を取得（NetEase、QQ Music、LRCLIB）、行ごとのスクロールと手動検索をサポート
- **AI翻訳** - 3つの翻訳スタイル（忠実、詩的、機械翻訳）、Gemini搭載
- **ノートシステム** - すべての曲にノートと評価を追加、あなたの音楽ストーリーを記録
- **タイムマシン** - 過去の今日に保存した曲を再発見、忘れられた音楽の思い出
- **音楽インサイト** - AIがリスニング習慣を分析し、パーソナライズされた音楽パーソナリティを生成
- **ポスター生成** - 歌詞、ノート、アルバム用の美しい共有ポスターを作成
- **ダイナミックテーマ** - アルバムアートから自動的にカラースキームを生成
- **多言語対応** - 英語、簡体字中国語、繁体字中国語、日本語

## 必要条件

- **Spotify Premium** - フル機能（再生コントロール、デバイス切り替えなど）には Premium サブスクリプションが必要
- **無料ユーザー** - 歌詞閲覧、ログイン、ノート記録は可能ですが、再生コントロールはできません
- **Google AI API Key**（オプション）- 歌詞翻訳とAIインサイト機能用、無料で取得可能
