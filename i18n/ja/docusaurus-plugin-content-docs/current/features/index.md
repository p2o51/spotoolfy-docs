# 機能概要

Spotoolfy は Spotify 体験を向上させる豊富な機能を提供しています。利用可能なすべての機能の概要をご覧ください。

## メインページ

Spotoolfy には3つのメインナビゲーションページがあります：

### [再生中ページ](./now-playing)
現在の再生を表示するコアインターフェース：
- **プレーヤー**：アルバムカバー、再生コントロール、プログレスバー
- **歌詞**：リアルタイム歌詞表示、翻訳、歌詞コピーモード
- **ノート**：ステータスパネル、ノートリスト、関連ノート
- **キュー**：再生キューの曲
- **フローティングアクションボタン**：クイックノート追加エントリー

### [ライブラリページ](./library)
音楽コレクションセンター：
- **カルーセル**：最近再生したアルバム/プレイリスト（15個）
- **インサイト**：AI生成の音楽パーソナリティと分析
- **検索**：トラック、アルバム、アーティスト、プレイリストのSpotifyグローバル検索

### [探索ページ](./explore)
音楽の旅と思い出：
- **タイムマシン**：過去の今日に保存した曲
- **ランダムレビュー**：過去のノートをランダム表示
- **すべてのノート**：時間順に並べられた完全なノートリスト
- **フィルター＆検索**：評価でフィルター、キーワード検索

### [設定とデバイス](./settings)
設定とデバイス管理：
- **APIキー**：Google AI、Spotify設定
- **翻訳設定**：言語、スタイル、深層翻訳
- **データ管理**：インポート、エクスポート
- **デバイス管理**：Spotifyデバイスの表示と切り替え

## コア機能

### リアルタイム歌詞
- 複数ソースの歌詞：NetEase、QQ Music、LRCLIB
- 現在再生中の曲に自動マッチ
- 行ごとにハイライトスクロール
- 歌詞をタップしてシーク
- 見つからない時は手動検索

### AI翻訳
Google Gemini搭載の3つの翻訳スタイル：
- **忠実**：単語ごとの正確な翻訳
- **詩的**：創造的で芸術的な解釈
- **機械**：初期の機械翻訳をシミュレート

NetEaseネイティブ翻訳もサポート（利用可能な場合）。

### ノートシステム
- 任意のトラックに評価を追加（👎 嫌い / 😐 普通 / 🔥 好き）
- 個人的な感想と思い出を記録
- 歌詞位置を「歌詞スナップショット」として自動記録
- 評価変化の傾向を表示
- 関連ノートを発見（同じ曲の異なるバージョン）

### タイムマシン
- 過去の今日に保存した曲を発見
- 忘れられた音楽の思い出を再体験
- カスタム日付範囲クエリ
- スパースインデックス技術で最適化

### 音楽インサイト
- AIが最近再生した音楽を分析
- パーソナライズされた「音楽パーソナリティ」タグを生成
- ムード分析と曲のおすすめ
- Google Gemini搭載

### ポスター生成
美しい共有画像を作成：
- 曲情報付き歌詞ポスター
- メモリーカードとしてのノートポスター
- アルバムカバーポスター
- ワンタップで保存または共有

### その他の機能
- **ダイナミックテーマ**：アルバムアートからの色
- **ダークモード**：システムダークモードサポート
- **多言語**：英語、中国語（簡体字/繁体字）、日本語
- **データエクスポート**：すべてのノートと翻訳をエクスポート
- **キュープレビュー**：次に再生される曲を確認

## 再生コントロール

フルSpotify再生コントロール（Premiumが必要）：
- 再生、一時停止、前、次
- シャッフルとリピートモード
- プログレスシーク
- デバイス切り替え
- 音量コントロール（対応デバイス）

## レスポンシブデザイン

Spotoolfy は異なる画面サイズに対応：
- **スマートフォン**：最適化された単一カラムレイアウト
- **タブレット**：サイドバイサイドレイアウト、マルチカラムグリッド
- **Web**：読みやすさのため最大1200pxコンテンツ幅
