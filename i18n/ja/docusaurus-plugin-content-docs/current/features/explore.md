# 探索ページ

探索ページは音楽ノートセンターで、タイムマシン、ランダムレビュー、完全なノートリストを含みます。ここに音楽とのすべてのストーリーが記録されています。

## タイムマシン

タイムマシンは過去の今日に保存した曲を再発見し、忘れられた音楽の思い出を発見できます。

### 今日の履歴
- 過去の今日（±1日の許容範囲）にお気に入りに追加した曲を自動的に検索
- 「X年前の今日」ラベルを表示
- 曲名、アーティスト、アルバムカバーを表示

### タイムマシンインタラクション
- **曲をタップ**：メモリー詳細を開き、完全な情報を表示
- **再生**：過去のこの曲を再生
- **すべて表示**：完全なタイムマシンページに入る

### タイムマシンページ
「すべて表示」または日付範囲ボタンをタップしてタイムマシンページに入る：
- マッチするすべてのメモリー曲を表示
- カスタム日付範囲クエリをサポート
- 日付ボタンをタップして「今日の履歴」と「カスタム範囲」モードを切り替え

### スパースインデックス
タイムマシンはパフォーマンス最適化のためにスパースインデックス技術を使用：
- 初回使用時にSpotifyライブラリをスキャン
- インデックス構築後はクエリが非常に高速
- インデックスデータはローカルに保存、アップロードなし

## ランダムレビュー

過去に書いたノートをランダムに表示し、音楽関連の思い出を再体験できます。

### 自動カルーセル
- 4秒ごとに異なるノートを自動的に切り替え表示
- ノート内容、曲情報、評価を表示

### レビューカード
カードに表示：
- ノート内容（ある場合）
- 歌詞スナップショット（ある場合）
- 曲名、アーティスト名
- アルバムカバー
- 評価アイコン
- 記録時間

### インタラクション
- **カードをタップ**：ノート詳細を開く
- **カードを長押し**：その曲を即座に再生

## カルーセルエリア

タイムマシンとランダムレビューはスワイプ可能なカルーセルエリアを形成：
- **1ページ目**：タイムマシン（今日の履歴）
- **2ページ目**：ランダムレビュー
- 下部にページインジケーター
- 左右スワイプで切り替え

## すべてのノート

ページ下部にはすべての記録されたノートが時間の逆順で表示されます。

### ノート検索
検索バーでノートを検索：
- ノート内容の検索をサポート
- 曲名の検索をサポート
- アーティスト名の検索をサポート

### フィルター機能
フィルタータグでノートをすばやくフィルター：

| フィルター | 説明 |
|----------|------|
| **すべて** | 内容のあるすべてのノートを表示 |
| **🔥 好き** | 「好き」と評価されたノートのみ表示 |
| **😐 普通** | 「普通」と評価されたノートのみ表示 |
| **👎 嫌い** | 「嫌い」と評価されたノートのみ表示 |
| **⭐ 評価のみ** | テキスト内容のない評価記録のみ表示 |

各フィルタータグの横に対応するノート数が表示されます。

### ノートカード
各ノートに表示：
- **ノート内容**：書いたテキスト（ある場合）
- **歌詞スナップショット**：記録時の歌詞（ある場合）
- **曲情報**：曲名、アーティスト、アルバムカバー
- **記録時間**：「今日 HH:mm」「昨日 HH:mm」または「MM-DD HH:mm」で表示
- **評価アイコン**：👎 / 😐 / 🔥

### ノート操作

#### ノートをタップ
ノート詳細パネルを開き、以下が可能：
- 完全なノート内容を表示
- 評価を変更
- ノート内容を編集
- 曲を再生（記録されたタイムスタンプ位置から）
- 共有ポスターを生成
- ノートを削除

#### スワイプ操作（狭い画面モード）
- **右スワイプ**：曲を再生
- **左スワイプ**：ノートを削除（確認が必要）

### 統計情報
フィルターバーの左側にノート総数が表示され、どれだけの音楽ストーリーを記録したかがわかります。

## データリフレッシュ

ページを下に引いてデータをリフレッシュ：
- ノートリストを再読み込み
- タイムマシンデータを更新
- ランダムレビューをリフレッシュ

## レスポンシブレイアウト

探索ページは画面幅に基づいてレイアウトを調整：

### 狭い画面（スマートフォン）
- 単一カラムノートリスト
- スワイプ操作をサポート（左スワイプで削除、右スワイプで再生）
- よりコンパクトなカードデザイン

### 広い画面（タブレット/デスクトップ）
- 2カラムマソンリーレイアウト
- カードグリッド表示
- セパレーター装飾

## 空の状態

まだノートを記録していない場合：
- フレンドリーな空の状態プロンプトを表示
- 「再生中」ページで最初のノートを追加するようガイド
