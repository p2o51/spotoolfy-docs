# 設定とデバイス

右上のユーザーアイコンをタップして設定ページを開きます。ここでSpotoolfyのすべての設定オプションを管理できます。

## アカウント設定

### Spotifyアカウント
- **ログイン/ログアウト**：Spotifyアカウント接続を管理
- 現在ログインしているユーザー名を表示
- ログインには完全な再生コントロール機能のためSpotify Premiumが必要

### Google AI API Key
- Gemini翻訳とインサイト機能用
- タップしてAPI Keyを入力または変更
- API Keyはローカルデバイスに安全に保存

**API Keyの取得**：
1. [Google AI Studio](https://aistudio.google.com/apikey)にアクセス
2. Googleアカウントでログイン
3. 新しいAPI Keyを作成
4. Spotoolfyにコピー＆ペースト

:::note
個人的な能力の制限により、現在Gemini 2.5 Flashシリーズモデル用にプロンプトのみ最適化されています。他のモデルはサポートされていません。必要に応じて自分でコンパイルして変更してください。
:::

## 言語設定

### Gemini's Language（翻訳言語）
AI翻訳とインサイト出力のターゲット言語を選択：
- **English** - 英語
- **简体中文** - 簡体字中国語
- **繁體中文** - 繁体字中国語
- **日本語** - Japanese

この設定はアプリインターフェース言語とは独立しています。以下が可能：
- 中国語インターフェース + 英語翻訳
- 英語インターフェース + 中国語翻訳
- その他の任意の組み合わせ

### アプリ言語
アプリインターフェース言語はデフォルトでシステム設定に従うか、システム設定でSpotoolfy用に個別に設定できます。

## 翻訳設定

### 翻訳スタイル
好みの歌詞翻訳スタイルを選択：

| スタイル | 説明 | 特徴 |
|---------|------|------|
| **Faithful** | 忠実翻訳 | 原文の意味を正確に伝え、構造を保持 |
| **Melodramatic Poet** | 詩的翻訳 | より詩的な言語を使用、感情表現を強化 |
| **Machine Classic** | 機械翻訳 | 初期の機械翻訳の直訳スタイルをシミュレート、時に興味深い結果 |
| **NetEase Provider** | 網易雲翻訳 | 網易雲音楽のネイティブ翻訳を使用（利用可能な場合） |

### 歌詞を1行としてコピー
有効にすると、歌詞や翻訳をコピーする際に改行をスペースに置き換え、以下に適しています：
- ソーシャルメディアへの共有
- 単一行入力フィールドへの貼り付け
- フォーマットの混乱を回避

### 深層翻訳

:::warning 非推奨
処理時間が長すぎるため、この機能は v1.3 で削除されました。
:::

- Gemini 2.5 Flashの思考バジェット（1024 Tokens）を設定
- より高品質で深い翻訳を提供する可能性
- **注意**：APIクォータを多く消費、有料API Keyを使用している場合は使用量を監視
- 価格は[Google AIドキュメント](https://ai.google.dev/pricing)を参照

## データ管理

### データエクスポート
すべてのローカルデータをJSONファイルとしてエクスポート：
- すべてのノート記録
- 曲情報
- 翻訳キャッシュ
- 再生コンテキスト記録

エクスポートファイルの用途：
- データバックアップ
- 他のデバイスへの移行
- データ分析

### データインポート
以前エクスポートしたJSONファイルからデータをインポート：
- データは既存データとマージ
- 既存のノートを上書きしない
- インポート後すぐに有効

### インポート/エクスポートプロセス
1. **エクスポート**：設定 → Export Data → 保存場所を選択 → 保存
2. **インポート**：設定 → Import Data → JSONファイルを選択 → 確認

## デバイス管理

右上のデバイスアイコン（🔊）をタップしてデバイス管理ページを開きます。

### デバイスリスト
利用可能なすべてのSpotifyデバイスを表示：
- デバイス名
- デバイスタイプアイコン（スマートフォン、コンピュータ、スピーカーなど）
- 現在のアクティブ状態

### デバイスタイプ
Spotoolfyがコントロールできるデバイスタイプ：
- **スマートフォン/タブレット**：iOS、Androidデバイス
- **コンピュータ**：Windows、macOS上のSpotifyクライアント
- **スマートスピーカー**：Spotify Connectをサポートするスピーカー
- **テレビ**：Spotifyをサポートするスマートテレビ
- **車載**：Spotifyをサポートする車載システム

### デバイス切り替え
リスト内のデバイスをタップして再生ターゲットを切り替え：
- 音楽は新しいデバイスにシームレスに転送
- 再生進行は同期を維持
- 一部のデバイスでは先にそのデバイスでSpotifyを開く必要がある場合があります

### デバイスリフレッシュ
デバイスリストは15秒ごとに自動リフレッシュ、または下に引いて手動リフレッシュ。

## よくある質問

### デバイスリストが空？
- Spotifyクライアントが実行中であることを確認
- Spotifyで曲を再生してデバイスをアクティブにしてみる
- デバイスが同じネットワークに接続されているか確認

### API Keyが動作しない？
- コピー時に余分なスペースがないことを確認
- API Keyを再生成してみる
- APIクォータが使い果たされていないか確認

### 翻訳品質が理想的でない？
- 翻訳スタイルを切り替えてみる
- 深層翻訳を有効にする
- 翻訳品質はソース歌詞の正確性に依存

### データエクスポートに失敗？
- ストレージ権限を確認
- 十分なストレージ容量があることを確認
- 別の保存場所を選択してみる
