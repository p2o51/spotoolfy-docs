# インストールとコミュニティ

## インストールガイド

### Android インストール

#### 方法1：Google Play クローズドテスト（推奨）

1. [Google Groups テストグループ](https://groups.google.com/g/spotoolfytest)に参加
2. [Google Play ストアページ](https://play.google.com/store/apps/details?id=com.gojyuplusone.spotoolfy.spotoolfy_flutter)にアクセス
3. 「インストール」をタップ

:::note
クローズドテスト版にアクセスするには、まず Google Groups に参加する必要があります。
:::

#### 方法2：APK 直接ダウンロード

1. [Discord コミュニティ](https://discord.gg/v56yDs6vRN)に参加
2. `#releases` チャンネルから最新の APK をダウンロード
3. デバイスに APK をインストール（「不明なソース」の許可が必要な場合があります）

### iOS/iPadOS インストール

1. [TestFlight テストリンク](https://testflight.apple.com/join/h2GR2Gbf)にアクセス
2. TestFlight がインストールされていない場合、App Store からインストールを促されます
3. TestFlight でテスト招待を承諾し、Spotoolfy をインストール

### Web版（ベータ）

- https://app.spotoolfy.gojyuplus.com/ にアクセス
- Web版は機能が制限されています：
  - リモート再生デバイスのコントロール不可
  - Web API 機能のみサポート
  - 歌詞閲覧、ノート記録、翻訳は利用可能

## システム要件

| プラットフォーム | 最小バージョン |
|-----------------|---------------|
| Android | Android 6.0 (API 23) |
| iOS | iOS 12.0 |
| Web | モダンブラウザ（Chrome、Firefox、Safari、Edge） |

## 問題の報告

問題が発生した場合は：
1. [FAQ・既知の問題](../advanced/faq)を確認
2. Discord の `#feedback` または `#bug-reports` チャンネルに投稿
3. GitHub で Issue を提出

報告時に提供してください：
- デバイスモデルとシステムバージョン
- Spotoolfy バージョン番号
- 問題の詳細説明と再現手順
- 可能であればスクリーンショットや画面録画
