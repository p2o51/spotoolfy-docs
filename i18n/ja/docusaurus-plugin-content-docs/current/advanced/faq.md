# FAQ・既知の問題

## 既知の問題

### 歌詞関連

#### 歌詞が正しく表示されない
- **症状**：歌詞のタイムスタンプが音楽と同期していない、または歌詞の内容が間違っている
- **原因**：サードパーティの歌詞ソースにエラーがある可能性
- **解決策**：手動検索機能を使用して、他のバージョンの歌詞を選択

#### 一部の曲に歌詞がない
- **症状**：「歌詞が見つかりません」と表示
- **原因**：曲がマイナーなトラック、または曲情報が不完全な可能性
- **解決策**：異なるキーワードの組み合わせで手動検索を試す

#### 翻訳の読み込みが遅い
- **症状**：翻訳をタップ後、長時間待機
- **原因**：Gemini APIの応答時間がネットワークの影響を受ける
- **解決策**：辛抱強く待つ、翻訳結果はキャッシュされ、次回は即座に読み込み

### 再生コントロール

#### デバイスリストが空
- **症状**：Spotifyデバイスが見つからない
- **原因**：アクティブなSpotifyクライアントがない
- **解決策**：
  1. 任意のデバイスでSpotifyクライアントを開く
  2. 曲を再生してデバイスをアクティブにする
  3. Spotoolfyでデバイスリストをリフレッシュ

#### 再生コントロールが応答しない
- **症状**：再生/一時停止をタップしても効果がない
- **原因**：トークンの期限切れまたはネットワークの問題の可能性
- **解決策**：
  1. ネットワーク接続を確認
  2. Spotifyに再ログインしてみる
  3. Spotifyクライアントが実行中であることを確認

#### トラック変更後にプログレスバーが同期しない
- **症状**：プログレスバーの表示が実際の再生位置と一致しない
- **原因**：ネットワーク遅延による状態更新の遅れ
- **解決策**：数秒待つと、進行が自動的に同期

### タイムマシン

#### 初回のタイムマシン使用が遅い
- **症状**：タイムマシンを初めて開く時に長時間待機
- **原因**：スパースインデックスを構築中、Spotifyライブラリをスキャン
- **解決策**：辛抱強く待つ、インデックス構築後はクエリが非常に高速

#### タイムマシンにコンテンツが表示されない
- **症状**：「メモリーが見つかりません」と表示
- **原因**：過去の今日に曲を保存していない可能性
- **解決策**：より大きな時間範囲をクエリするためにカスタム日付範囲を使用してみる

### iOS固有の問題

#### ログイン後にアプリに戻れない
- **症状**：Spotify認証完了後もブラウザに留まる
- **原因**：URLスキームコールバックがシステムによってインターセプトされた可能性
- **解決策**：手動でSpotoolfyアプリに切り替え

#### TestFlightバージョンの期限切れ
- **症状**：アプリが開けない、バージョン期限切れと表示
- **原因**：TestFlightビルドには有効期限の制限あり
- **解決策**：新しいバージョンのリリースを待つ、または開発者に更新を連絡

## よくある質問

### アカウントとログイン

**Q: Spotify Premiumは必要ですか？**
A: フル機能にはPremiumが必要です。無料ユーザーは歌詞閲覧、ノート記録は可能ですが、再生コントロールはできません。

**Q: Spotifyデータは安全ですか？**
A: Spotoolfyはパスワードを保存しません。OAuth認証は必要な権限のみを取得し、トークンはローカルデバイスに安全に保存されます。

**Q: 複数のアカウントに同時にログインできますか？**
A: 現在、単一アカウントログインのみサポート。アカウントを切り替えるには、まず現在のアカウントからログアウトしてください。

### 歌詞と翻訳

**Q: 一部の歌詞が文字化けしているのはなぜ？**
A: 歌詞ソースのエンコーディングの問題の可能性。他のソースを手動検索してみてください。

**Q: 翻訳にお金がかかりますか？**
A: Google AI API Keyは無料で取得・使用可能、個人使用は完全無料です。

**Q: 翻訳された歌詞を保存できますか？**
A: 翻訳は自動的にローカルにキャッシュされます。コピー機能を使用して保存することもできます。

**Q: 翻訳スタイルの違いは？**
A:
- Faithful：直訳、原文の意味を正確に伝える
- Melodramatic Poet：意訳、より詩的で感情的
- Machine Classic：初期の機械翻訳をシミュレート、独特のスタイル
- NetEase：網易雲のネイティブ翻訳を使用

### ノートとデータ

**Q: ノートはクラウドに同期されますか？**
A: 現在、ノートはローカルデバイスにのみ保存。エクスポート機能を使用してデータをバックアップできます。

**Q: スマートフォンを変更するとノートは失われますか？**
A: 手動でエクスポートしてからインポートする必要があります。定期的にデータをバックアップすることをお勧めします。

**Q: すべてのデータを削除できますか？**
A: デバイスストレージでアプリデータをクリアするか、アンインストールして再インストールできます。

**Q: ノートに文字数制限はありますか？**
A: ハードリミットはありませんが、簡潔に保つことをお勧めします。

### 機能と互換性

**Q: Apple Watchはサポートされていますか？**
A: 現在サポートされていませんが、Spotoolfyを通じてiPhone/iPadをコントロールできます。

**Q: オフラインで使用できますか？**
A: 歌詞と翻訳にはネットワークが必要。キャッシュされたコンテンツはオフラインで閲覧可能ですが、再生コントロールにはインターネットが必要です。

**Q: ポッドキャストはサポートされていますか？**
A: 現在主に音楽用に最適化されており、ポッドキャストでは歌詞を取得できない場合があります。

**Q: なぜAndroidウィジェットがないのですか？**
A: ウィジェット機能は計画中で、将来のバージョンで追加予定です。

### 問題報告

**Q: バグを報告するには？**
A:
1. Discord `#bug-reports` チャンネル（推奨）
2. GitHub Issues
3. メール lastnatsu51@gmail.com

**Q: 新機能をリクエストするには？**
A:
1. Discord `#feedback` チャンネルで議論
2. GitHub IssuesでFeature Requestを提出

**Q: 問題報告時に何を提供すればよいですか？**
A:

- Spotoolfyバージョン番号
- 問題の詳細説明
- 再現手順
- スクリーンショットや画面録画（可能であれば）

## エラーコードリファレンス

| エラーコード | 意味 | 解決策 |
|-------------|------|--------|
| `AUTH_EXPIRED` | 認証の期限切れ | Spotifyに再ログイン |
| `NO_ACTIVE_DEVICE` | アクティブなデバイスがない | Spotifyクライアントで任意の曲を再生 |
| `NETWORK_ERROR` | ネットワークエラー | ネットワーク接続を確認 |
| `LYRICS_NOT_FOUND` | 歌詞が見つからない | 手動検索を試す |
| `API_RATE_LIMITED` | APIリクエストが頻繁すぎる | 後で再試行 |
| `TRANSLATION_FAILED` | 翻訳に失敗 | API Key設定を確認 |
