# Spotoolfy Documentation

Official documentation for [Spotoolfy](https://github.com/p2o51/spotoolfy_flutter) - a Spotify companion client with lyrics, AI translation, notes, and time machine features.

**Live Documentation**: https://docs.spotoolfy.gojyuplus.com

## Development

This documentation is built using [Docusaurus](https://docusaurus.io/).

### Installation

```bash
npm install
```

### Local Development

```bash
npm run start
```

This starts a local development server at `http://localhost:3000`. Changes are reflected live.

### Build

```bash
npm run build
```

This generates static content in the `build` directory.

### Deployment

The documentation is automatically deployed to the live site on push to main branch.
