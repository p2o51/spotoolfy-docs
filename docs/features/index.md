# Features Overview

Spotoolfy is packed with features to enhance your Spotify experience. Here's a comprehensive overview of what you can do.

## Main Pages

Spotoolfy has three main navigation pages:

### [Now Playing Page](./now-playing)
The core interface showing current playback with:
- **Player**: Album cover, playback controls, progress bar
- **Lyrics**: Real-time lyrics display, translation, copy lyrics mode
- **Notes (Records)**: Status panel, notes list, related notes
- **Queue**: Upcoming songs in playback queue
- **Floating Action Button**: Quick note adding entry

### [Library Page](./library)
Your music collection center with:
- **Carousel**: Recently played albums/playlists (15 items)
- **Insights**: AI-generated music personality and analysis
- **Search**: Spotify global search for tracks, albums, artists, playlists

### [Explore Page](./explore)
Your music journey and memories:
- **Time Machine**: Songs you saved on this day in previous years
- **Random Review**: Random display of historical notes
- **All Notes**: Complete notes list sorted by time
- **Filter & Search**: Filter by rating, search by keywords

### [Settings & Devices](./settings)
Configuration and device management:
- **API Keys**: Google AI, Spotify configuration
- **Translation Settings**: Language, style, deep translation
- **Data Management**: Import, export data
- **Device Management**: View and switch Spotify devices

## Core Features

### Real-time Lyrics
- Multi-source lyrics: NetEase, QQ Music, LRCLIB
- Auto-match current playing song
- Line-by-line highlight scrolling
- Tap lyrics to seek
- Manual search when needed

### AI Translation
Three translation styles powered by Google Gemini:
- **Faithful**: Word-for-word accurate translation
- **Melodramatic Poet**: Creative, artistic interpretation
- **Machine Classic**: Simulates early machine translation

Plus NetEase native translation support when available.

### Time Machine
- Discover songs saved on this day in previous years
- Relive forgotten music memories
- Custom date range queries
- Optimized with sparse index technology

### Music Insights
- AI analyzes your recently played music
- Generates personalized "music personality" tags
- Mood analysis and song recommendations
- Powered by Google Gemini

### Poster Generation
Create beautiful shareable images:
- Lyrics posters with song info
- Note posters as memory cards
- Album cover posters
- One-tap save or share

### Other Features
- **Dynamic Theme**: Colors from album art
- **Dark Mode**: System dark mode support
- **Multi-language**: English, Chinese (Simplified/Traditional), Japanese
- **Data Export**: Export all notes and translations
- **Queue Preview**: See upcoming songs

## Playback Control

Full Spotify playback control (Premium required):
- Play, pause, previous, next
- Shuffle and repeat modes
- Progress seeking
- Device switching
- Volume control (on supported devices)

## Responsive Design

Spotoolfy adapts to different screen sizes:
- **Phone**: Optimized single-column layouts
- **Tablet**: Side-by-side layouts, multi-column grids
- **Web**: Maximum 1200px content width for readability
