# Installation & Community

## Installation Guide

### Android Installation

#### Option 1: Google Play Closed Testing (Recommended)

1. Join the [Google Groups Testing Group](https://groups.google.com/g/spotoolfytest)
2. Visit the [Google Play Store page](https://play.google.com/store/apps/details?id=com.gojyuplusone.spotoolfy.spotoolfy_flutter)
3. Click "Install"

:::note
You must join the Google Groups first to access the closed testing version.
:::

#### Option 2: Direct APK Download

1. Join the [Discord Community](https://discord.gg/v56yDs6vRN)
2. Download the latest APK from the `#releases` channel
3. Install the APK on your device (may require enabling "Unknown Sources")

### iOS/iPadOS Installation

1. Visit the [TestFlight testing link](https://testflight.apple.com/join/h2GR2Gbf)
2. If TestFlight is not installed, the App Store will prompt you to install it
3. Accept the test invitation in TestFlight and install Spotoolfy

### Web Version (Beta)

- Visit https://app.spotoolfy.gojyuplus.com/
- Web version has limited features:
  - Cannot control remote playback devices
  - Only supports Web API functionality
  - Can view lyrics, take notes, and use translation

## Reporting Issues

When encountering issues, please:
1. Check the [FAQ & Known Issues](../advanced/faq)
2. Post in Discord's `#feedback` or `#bug-reports` channels
3. Submit an Issue on GitHub

When reporting, please provide:
- Device model and system version
- Spotoolfy version number
- Detailed description of the issue and reproduction steps
- Screenshots or screen recordings if possible
